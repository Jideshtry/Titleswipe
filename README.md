# TitleSwipe™ — AI Real Estate Title Due Diligence Software

> Real estate title opinions in your law firm's own templates. In hours, not days.

## Deploy to Your Titleswipe.com in 3 Steps

### 1. Push to GitHub
```bash
git init
git add .
git commit -m "TitleSwipe website — complete rebrand, SEO, logo"
git branch -M main
git remote add origin https://github.com/Jideshtry/Titleswipe.git
git push -u origin main --force
```

### 2. Connect Vercel
- Go to vercel.com/new
- Import the Titleswipe GitHub repository
- Click Deploy — no settings needed, vercel.json handles everything

### 3. Set Custom Domain
- Vercel Dashboard → Settings → Domains → Add www.titleswipe.com
